# Deep_Learning_Spring2026_Final

The Jupyter Notebook "DL_final_final_version.ipynb" is the code used to train the model. The entirety of the notebook was run in a Google Colab environment. The files for the data in my case was on Google Drive and therefore the notebook will attempt to mount the drive to the current runtime. Alternatively, the files can be uploaded from your local device to the runtime environment. It should be noted that the variables for the directories should be modified as needed.

The model weights can be found in the folder "science_qa_model_v6".
